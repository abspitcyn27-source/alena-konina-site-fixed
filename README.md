# Alena Konina — Nails (Moscow) + Academy (Scaffold)

This is a multilingual (RU/EN) Next.js site focused on client bookings in Moscow (m. Ulitsa 1905 Goda),
with an "Academy" section scaffolded for future course sales.

## Quick start
1) Install Node.js 18+  
2) In terminal:
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Deploy to Vercel (recommended)
Option A (easy): GitHub import
1) Create a new GitHub repo
2) Push this project
3) In Vercel: Add New → Project → Import that repo → Deploy

Option B: Vercel CLI
```bash
npm i -g vercel
vercel login
vercel
vercel --prod
```
